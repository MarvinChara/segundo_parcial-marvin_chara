Sobre el Proyecto

Este proyecto fue desarrollado con Angular y Bootstrap, como parte del segundo parcial de Tecnologías de Internet.

Se implementó un template gratuito (GP Template de BootstrapMade) y se integraron frameworks modernos como Angular 20 para construir una aplicación modular y escalable.

Incluye prácticas de PWA, notificaciones, y despliegue en GitHub Pages, cumpliendo los requerimientos del examen.

Servicios

Algunas tecnologías aplicadas en este proyecto:

Angular 20

Framework de desarrollo frontend moderno basado en TypeScript.

Bootstrap 5

Framework CSS para diseño responsive y componentes visuales listos.

PWA

Aplicación web progresiva con soporte offline y notificaciones.
